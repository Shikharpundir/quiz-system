# quiz-system
A simple and interactive Quiz System web app built using Flask. Users can register, log in, attempt multiple-choice quizzes, and view scores instantly. Includes a leaderboard for competition and an admin panel to manage questions and users. Designed with a responsive UI using HTML, CSS, and Bootstrap.
